# M365 Domain Removal Guide

## Introduction
This guide provides step-by-step instructions for safely removing a domain from Microsoft 365. 

## Steps to Remove a Domain
1. Sign in to the Microsoft 365 admin center.
2. Go to **Setup** > **Domains**.
3. Select the domain you wish to remove.
4. Ensure no users or resources are using the domain.
5. Follow the on-screen instructions to remove the domain.

## Conclusion
Always double-check before removal to ensure that the domain is no longer in use and to maintain service continuity.
